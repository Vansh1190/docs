## Seller Side AdNet Campaign Manager
This is a seller side campaign manager for AdNet. It is built using Next.js.
and it is live on https://adnet2.vercel.app/

About AdNet.
A seller can do Signup using `bpp_id & bpp_url`, after verification, seller can create various stores and we will load their Products and Menus


A Seller can create different `campaigns` for different stores and can track the performance of every campaign with various customization options for their ads.

#### Theme
- We have used [Minimal.cc](https://minimals.cc/) theme for this project which is based on [Material UI](https://mui.com/).


#### Tech Stack
- Next.js
- MongoDB
- Razorpay

## Menu Features
> 1. Dashboard 
>       - It contains various metrics and insights about the Store.
>       - We can create new campaigns from here.

> 2. Campaigns
>       - A seller can view all the campaigns they have created.
>           - `Active Campaigns` A seller can view the active campaigns.
>           - `Upcoming Campaigns` A seller can view the upcoming campaigns.
>           - `Past Campaigns` A seller can view the past campaigns.
>       - A seller can view the performance and insights of every campaign by clicking on the campaign.
> 3. Insights
>       - A seller can view the insights of their stores with various options like today, weekly, monthly etc.
> 4. Profile
>       - A seller can view and can update their profile.
>       - A seller can view and can update their stores.
>       - A seller can add new or delete existing stores.
>       - A seller can manage their payment details.
